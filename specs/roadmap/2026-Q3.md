## Em Andamento — Q3 Semanas 10-14

### FEATURE-006: Layout Dual-Faixa — Estado Atual vs. Proposta de Mudança (P1)
*Separar visualmente o que o arquivo é do que vai se tornar*

- [ ] Faixa azul (read-only): Nome atual / Formato / Título atual / Autor atual / ISBN / Ano / Editora atual
- [ ] Faixa verde (editável): Novo Nome / Novo Título / Novo Autor / Novo Ano / Nova Editora
- [ ] Coluna ⬤ com indicador de qualidade por linha (🟢🟡🔴)
- [ ] Cabeçalho de grupo duplo sobre as colunas
- [ ] Badges de origem por célula (PDF / OL / GB / ✎ manual)
- [ ] Estados de célula: âmbar (sugerido) / branco (vazio) / verde (confirmado) / vermelho (erro)
- [ ] `GroupedHeaderView` (QHeaderView customizado com 2 linhas de cabeçalho)
- [ ] `FileRow` dataclass substituindo o dict atual no `FileTableModel`
- [ ] Botões: Confirmar Linha / Limpar Proposta / Confirmar Todos

**Exit criteria:** usuário consegue ver claramente o que mudará antes de aplicar; zero regressões

---

### FEATURE-007: Workflow de Busca ISBN e Enriquecimento Completo (P1)
*Corrigir a busca para preencher todos os 5 campos e gravar metadados no PDF*

- [ ] `SearchPipeline` com 4 estratégias em cascata:
  1. ISBN embutido no PDF
  2. ISBN detectado no nome do arquivo (regex)
  3. Título + Autor extraídos do PDF
  4. Título + Autor inferidos do nome do arquivo (parser de padrão)
- [ ] `_parse_filename()` reconhece padrões: `SOBRENOME - Título (Ano)`, `Autor - Título`, ISBN-Título
- [ ] Busca preenche todos os 5 campos da faixa verde com badges de origem
- [ ] Normalização: título CAPS → Title Case; autor com pseudônimo → `Sobrenome, Nome (Pseudo)`
- [ ] Editora com lixo de metadado (Adobe, Microsoft, Acrobat) é descartada automaticamente
- [ ] Múltiplos resultados → linha expandível com até 3 alternativas rankeadas por confiança
- [ ] `pdf_metadata_writer.py` — write-back usando `doc.set_metadata()` + `doc.saveIncr()`
- [ ] Após apply: faixa azul atualizada com os novos metadados gravados
- [ ] `SearchWorker (QThread)` com progresso e cancelamento
- [ ] "Buscar Todos" pula linhas com faixa verde já confirmada (verde)

**Exit criteria:** "Buscar Todos" → faixa verde preenchida; "Apply Changes" → arquivo renomeado
E metadados gravados dentro do PDF; Ctrl+Z reverte rename e metadados
