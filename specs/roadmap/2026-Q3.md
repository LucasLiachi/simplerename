# Roadmap Q3 2026 — SimpleRename Book Organizer

## Theme: De Renomeador Genérico a Organizador de Biblioteca PDF

---

## Must Ship (P0) — Semanas 1-4

- [x] **EPIC-001** iniciado — visão e specs aprovadas
- [ ] **FEATURE-001**: Build Pipeline & Windows Installer automatizado via GitHub Actions
  - Entrega: `git tag v1.0.0` → instalador publicado automaticamente
  - Exit criteria: CI verde, `.exe` roda em Windows 10/11 limpo
- [ ] **FEATURE-002**: Extração automática de metadados de PDF (PyMuPDF + pypdf)
  - Entrega: colunas Título/Autor/ISBN/Ano preenchidas ao carregar pasta
  - Exit criteria: ≥ 70% dos PDFs com metadado embutido extraídos corretamente

---

## Should Ship (P1) — Semanas 5-9

- [ ] **FEATURE-003**: Busca online via Open Library + Google Books
  - Entrega: botão "Buscar Online" + busca em lote para linhas incompletas
  - Exit criteria: ≥ 80% de acerto por ISBN; cache local funcionando
- [ ] **FEATURE-004**: Estratégia CDD de catalogação e sugestão de pasta
  - Entrega: coluna "Pasta Sugerida" + preview da estrutura + "Apply with Folders"
  - Exit criteria: 10 categorias CDD mapeadas; mover arquivos funciona sem corrupção
- [ ] **FEATURE-005**: Planilha com Preview em tempo real + Undo/Redo (Ctrl+Z/Y)
  - Entrega: coluna Preview; HistoryManager conectado; fill handle sem conflito
  - Exit criteria: Ctrl+Z restaura arquivo em disco; zero regressões em tests/

---

## Nice to Have (P2/P3) — Backlog Q4

- [ ] Filtro de extensão na toolbar (`FilterSortManager` já implementado, só falta UI)
- [ ] Painel de histórico de operações (lista das últimas ações com timestamp)
- [ ] Suporte a EPUB e MOBI além de PDF
- [ ] Modo offline com dump local do Open Library (~30GB CSV, opcional)
- [ ] Code signing do executável Windows (certificado pago)
- [ ] Auto-update: checar nova versão ao abrir o app

## Capacity & Sequencing

```
Semana 1-2:  FEATURE-001 (build) ← desbloqueador de tudo
Semana 3-4:  FEATURE-002 (metadados locais) + limpeza de file_manager.py (ADR-002)
Semana 5-6:  FEATURE-003 (busca online)
Semana 7-8:  FEATURE-004 (catalogação CDD)
Semana 9:    FEATURE-005 (planilha + undo) + testes de regressão
```

### Dependências entre Features

```
FEATURE-001 (build)
    └── FEATURE-002 (metadados PDF)
            └── FEATURE-003 (busca online)
                    └── FEATURE-004 (catalogação)
    └── FEATURE-005 (planilha/undo)  ← paralelo com FEATURE-002
```
